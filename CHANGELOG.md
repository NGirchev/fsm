## [1.0.1] - 2025-12-09

### Added
- CI/CD pipeline with automated testing and code quality checks
- Code coverage reporting with JaCoCo (minimum 80% line coverage, 70% branch coverage)
- GitHub Actions workflow for continuous integration
- Project badges in README (Build Status, License, Kotlin Version)
- Comprehensive documentation sections (Features, Requirements, Versioning, Installation, Testing, CI/CD, Releasing)
- CONTRIBUTING.md with contribution guidelines
- CHANGELOG.md for tracking project changes

### Changed
- Improved README.md with detailed documentation
- Enhanced CI workflow with code quality checks (Detekt, Kotlinter)
- Added test coverage verification to build process

## [1.0.0] - Initial Release

### Added
- Basic FSM implementation
- Extended FSM implementation with domain support
- State transition table builder
- Event handling
- Guard conditions
- Actions and post-actions
- Exception handling for invalid transitions
